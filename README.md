## 強み
* Railsでの機能開発を始め、効率的なCI/CDを考慮したインフラ構築とリリース後の仮説検証のためのデータ可視化を一気通貫でゼロから進めることができる

* Sparkを活用した機械学習基盤を構築できるところ

* CircleCI導入やChatOpsを活用したデプロイの効率化や開発速度の向上ができるところ
